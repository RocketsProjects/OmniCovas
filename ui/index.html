About OmniCOVAS

What is OmniCOVAS?

OmniCOVAS is a local-first companion application for Elite Dangerous. It reads game data from your local machine to provide live command deck views, telemetry, and advisory features. It does not control your ship or automate gameplay. AI assistance is optional and is never treated as a source of game facts; core features keep working with no AI provider configured.

Public beta posture

OmniCOVAS is a public beta — not a v1.0 release. The following posture applies:

  • Local-first by default; no telemetry is sent to the maintainer.
  • Accessibility coverage is partial; full keyboard and screen-reader certification is deferred to a later certification pass.
  • External community data providers are disabled or require authorization by default, and are only used when you opt in.
  • Squadron multi-commander and shared state remain reserved for a later release; current Squadrons surfaces are local-only.
  • Meaningful actions are recorded in the Activity Log proof layer.
  • Builds are not yet code-signed; see the project SECURITY policy.
