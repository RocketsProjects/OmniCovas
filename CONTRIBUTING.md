# Contributing to OmniCOVAS

Thank you for your interest in OmniCOVAS.

OmniCOVAS is free-to-use source-available proprietary software. It is not open source.

Bug reports, feature requests, compatibility notes, documentation feedback, security reports, and other non-code feedback are welcome.

## Code Contributions

Code contributions, pull requests, patches, code snippets, and other technical contributions are not accepted unless separately agreed in writing or expressly accepted by Zakary Peters / RocketsProjects.

Do not submit code with the expectation that it will be reviewed, merged, redistributed, sublicensed, or accepted under public fork terms.

If code is accepted without a separate written agreement, the accepted contribution is governed by the contribution terms in `LICENSE.md` unless Zakary Peters / RocketsProjects expressly agrees otherwise in writing.

## Feedback

You may submit bug reports, feature requests, issue comments, discussions, compatibility reports, or other suggestions. Feedback may be used, copied, modified, published, distributed, and incorporated into OmniCOVAS or related materials without compensation.

Submitting feedback does not grant you any right to redistribute, publish, mirror, rehost, repackage, sublicense, sell, commercialize, or make available OmniCOVAS or modified versions.

## Security Reports

Report security issues privately to rocketsprojects.git@gmail.com. Do not open public issues for vulnerabilities.

## Permission Requests

For licensing, legal notices, sponsor-preview access, GitHub repository messaging, or permission requests, contact rocketsprojects.git@gmail.com.
